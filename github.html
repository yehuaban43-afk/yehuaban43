關於我

我是一位充滿熱情的前端開發者，專注於利用 HTML 和 CSS 創造優雅且功能強大的網頁體驗。我對「關注點分離」原則有深入理解，並致力於編寫清晰、可維護的程式碼。憑藉對最新網頁技術的敏銳洞察力，我能將設計理念轉化為響應式且用戶友好的介面。

我的專案

響應式作品集網站

負責從零開始構建一個完全響應式的個人作品集網站，展示我的前端技能。該網站採用現代設計風格，並優化了不同設備上的顯示效果。

互動式資料視覺化介面

開發了一個基於 D3.js 的互動式儀表板，用於呈現複雜的數據。專注於提升用戶體驗，確保數據的易讀性和互動性。
